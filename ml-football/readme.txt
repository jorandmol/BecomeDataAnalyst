"Aprendizaje automático aplicado al fútbol profesional" - Código

DESCRIPCIÓN DEL CONTENIDO
- data/ Directorio con archivos necesarios para reproducir los experimentos
- PlayerAnalysis.ipynb Archivo que contiene el segundo experimento
- Strategy.ipynb Archivo que contiene el primer experimento

PASOS PARA EJECUCIÓN
- Es necesaria una plataforma capaz de ejecutar archivos Jupyter Notebook, por lo que se recomienda la instalación de Anaconda
- Instalar las librerías de Python necesarias (por tanto, es requisito indispensable tener instalado ese lenguaje) mediante los siguientes comandos:
	pip install matplotlib
	pip install numpy
	pip install pandas
	pip install statsmodels
	pip install sklearn
	pip install yellowbrick
- Crear un directorio /visual donde se almacenan las imágenes del experimento
- *MUY IMPORTANTE* Clonar el repositorio de Statsbomb (disponible en https://github.com/statsbomb/open-data) en un directorio /open-data
- Descargar el fichero FCPython.py (disponible en https://github.com/Friends-of-Tracking-Data-FoTD/SoccermaticsForPython)
- Abrir los archivos Jupyter Notebook (.ipynb) y se ejecutar las celdas. Se recomienda que la ejecución de estas se realice de uno en uno para seguir correctamente las explicaciones
